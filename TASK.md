Микросервис для сокращения ссылок

Суть
Написать API для сервиса сокращения ссылок (аналог bitly)

Что должно быть

1. Эндпоинт POST /shorten принимает длинную ссылку, возвращает короткий идентификатор
2. Эндпоинт GET /{short_id} редиректит на оригинальную ссылку
3. Счетчик переходов по каждой ссылке хранится в БД и обновляется при каждом GET
4. Добавить эндпоинт GET /stats/{short_id} который возвращает количество переходов
5. Написать минимум 2 unit-теста на основную логику
